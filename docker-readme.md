# Быстрые команды для запуска компоуза

Сценарий 1: Полностью локальный запуск (Self-Hosted)
Поднимает базы, модели и приложение. Модели скачаются в том при первом запуске.

```bash
docker compose --profile self-hosted-ai --profile app up -d
```

Сценарий 2: Только базы данных
Для разработки, когда модели не нужны или используются внешние.

```bash
docker compose up -d
```

Сценарий 3: Локальные модели + отладка приложения на хосте
Поднимает базы и модели, но приложение запускаете через uv run на компьютере.

```bash
docker compose --profile self-hosted-ai up -d
```

Сценарий 4: Внешние API + Приложение
Поднимает базы и приложение. Приложение будет стучаться во внешний API (OpenAI/Anthropic), так как флаги SELF_HOSTED выключены.

```bash
docker compose --profile app up -d
```
